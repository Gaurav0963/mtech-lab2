Part 1: Python Fundamentals Solutions

1. Square each number in a list using `map` and `lambda`.

list(map(lambda x: x**2, numbers))

Explanation: The `lambda x: x**2` function squares its input `x`. `map` applies this function to every item in the `numbers` list.

2. Filter for even numbers in a list.

list(filter(lambda x: x % 2 == 0, numbers))

Explanation: The `lambda x: x % 2 == 0` function returns `True` only for even numbers. `filter` uses this to build a new list containing only the items for which the function returns `True`.

3. Output of the map code and a new similar question.

Output: `[20, 30, 40]`

New Question:

Given `words = ['hello', 'world', 'python']`, use `map` to get `[5, 5, 6]`.

list(map(len, words))

4. Rewrite a `for` loop using a list comprehension.

squared = [n**2 for n in numbers]

5. Sort a list of dictionaries by age.

sorted(people, key=lambda person: person['age'])

6. Difference between `map` and `filter`.

`map` applies a function to every item in an iterable and returns a new iterable with the results. `filter` returns a new iterable containing only the items from the original iterable for which a function returns `True`.

7. Create a `lambda` function to sum two arguments.

adder = lambda x, y: x + y
print(adder(5, 10))

8. List comprehension for numbers divisible by 3.

[num for num in range(1, 51) if num % 3 == 0]

9. What does a `lambda` function return?

A `lambda` function returns a function object.

10. Filter for strings with more than 5 characters.

list(filter(lambda word: len(word) > 5, words))

Part 2: EDA on Titanic Dataset Solutions

1. Load `titanic.csv` into a DataFrame.

import pandas as pd
df = pd.read_csv('titanic.csv')

2. Display the first 5 rows.

df.head()

3. Check for missing values.

df.isnull().sum()

4. Find the survival rate.

survival_rate = df['Survived'].mean() * 100
print(f"Survival Rate: {survival_rate:.2f}%")

5. Bar chart of passenger classes.

import seaborn as sns
import matplotlib.pyplot as plt

sns.countplot(x='Pclass', data=df)
plt.title('Passenger Count by Class')
plt.show()

6. Histogram of passenger ages.

sns.histplot(df['Age'].dropna(), bins=30, kde=True)
plt.title('Age Distribution of Passengers')
plt.show()

7. Survival rate by gender.

df.groupby('Sex')['Survived'].mean() * 100

Females had a much higher survival rate.

8. Survival rate by Sex and Pclass.

sns.catplot(x='Pclass', y='Survived', hue='Sex', data=df, kind='bar')
plt.title('Survival Rate by Sex and Pclass')
plt.show()

9. Fill missing 'Age' values.

A common strategy is to fill missing ages with the median age of the dataset. The median is often preferred over the mean as it's less sensitive to outliers.

median_age = df['Age'].median()
df['Age'].fillna(median_age, inplace=True)
